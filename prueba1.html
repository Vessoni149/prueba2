Comandos Git

  • git init : crea el repositorio

  • git congig --global user.gmail "mail desarrollador" : asi agregamos un desarrollador

  • git config --global user.name "nombre desarrollador" : se le asigna un nombre

  • git status : muestra el estado del proyecto dentro del repositorio

  • git add nombreDelProyecto.html : agrega un archivo al repositorio local que acabamos de crear

  • git comit : genera un snashot que sirve para el historial

    • se abrira una nueva consola donde vamos a ingresar un mensaje, para eso presionamos tecla "i", luego con esc se sale de esa funcion. Luego con el comando ":wq" se vuelve a la consola del git

  • git log :informa de los cambios que se realizaron, va a devolvernos un numero de comit
